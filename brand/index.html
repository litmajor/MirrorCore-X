Usage Guidelines

Logo Usage

  • Always maintain adequate clear space around the logo
  • Do not alter the logo colors or proportions
  • Use the horizontal logo for wide layouts
  • Use the icon for small sizes (favicons, app icons)

Color Usage

  • Primary cyan (#00D9FF) for interactive elements and highlights
  • Purple (#7B2FFF) for secondary emphasis and CTAs
  • Use gradients sparingly for premium feel
  • Maintain sufficient contrast for accessibility

Typography

  • Orbitron for the brand name and major headings
  • Rajdhani for section headings and subheadings
  • Inter for body text and UI elements
  • JetBrains Mono for code and technical content
